Pulling Code(Make sure you have git bash installed)

Open Git bash in VS code terminal

1. **Make sure you’re inside your project folder**  
   cd path/to/your/project


2. **Check your current branch**  
   git branch

3. **Pull the latest changes**  
   git pull origin main




Push your changes back to github

Make sure you saved your files
first time
1. git status
2. git add .
3. git commit -m "Your changes"
4. git push origin main

all after
1. git status
2. git add .
3. git commit -m "Your changes"
4. git push
